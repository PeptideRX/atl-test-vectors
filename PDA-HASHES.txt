Peptide Design Attestation (PDA) — Frozen Cross-Language Test Vectors
=====================================================================

These three SHA-256 hashes are the canonical 32-byte PDAs for the three
frozen test vectors. Any conforming implementation in any language MUST
produce these exact hex strings when run against the corresponding
input file.

If your implementation produces a different hash for any of these
vectors, your implementation is non-conforming.

These hashes are reproduced in three independent implementations:
  - Python  (private reference, pydantic v2 + sha2)
  - Rust    (private reference, serde_json + sha2)
  - TypeScript (public, github.com/PeptideRX/atl-verifier, Web Crypto)

The TypeScript implementation is open source. The Python and Rust
implementations are closed source for reasons documented at
github.com/PeptideRX (biosecurity-relevant code, patent-pending IP).
Anyone can run the TypeScript verifier against these vectors today.
